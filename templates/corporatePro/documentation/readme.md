## CUSTOMIZATION GUIDE
-------------------

### COLORS
To change the main colors, edit the CSS variables in the :root section of style.css:
- --primary-color: Main brand color
- --secondary-color: Secondary accent color
- --accent-color: Highlight and call-to-action color
- Update these variables to match your brand colors

### TEXT CONTENT
Edit text directly in the HTML files. Each page is clearly structured with comments to help you locate sections. Replace placeholder text with your actual content.

### IMAGES
Replace placeholder images with your own professional images:
1. Hero images in assets/images/hero/
2. Service images in assets/images/services/
3. Team photos in assets/images/team/
4. Blog images in assets/images/blog/
5. Logo files in assets/images/logos/

Ensure all images are optimized for web use to maintain fast loading times.

### ICONS
The template uses Font Awesome icons. To change an icon:
1. Visit fontawesome.com/icons
2. Find your desired icon
3. Copy the icon class (e.g., fas fa-chart-line)
4. Replace the existing icon class in the HTML

### RESPONSIVE ADJUSTMENTS
If you need to adjust the layout for specific screen sizes, edit the responsive.css file. The template uses a mobile-first approach, so styles build from small to large screens.

### JAVASCRIPT CUSTOMIZATION
The template uses vanilla JavaScript for all interactions. The main JavaScript file (script.js) is well-commented and organized. You can modify or extend functionality as needed.
