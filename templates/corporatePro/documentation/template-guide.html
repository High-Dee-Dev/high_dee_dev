Template Editing Guide

Step-by-step instructions for customizing your CorporatePro template

Welcome to Your Template!

This guide will help you customize the CorporatePro template even if you have limited coding experience. Each section explains how to modify different parts of the template.

Tip: Before making changes, always backup your original files. Work on a copy of the template to preserve the original version.

Changing Text Content

Follow these steps to update text throughout the template:

1 Open the HTML File

Open the HTML file you want to edit (e.g., index.html) in a code editor like Visual Studio Code, Sublime Text, or even Notepad.

2 Find the Text to Change

Look for the text you want to change. Each section is clearly marked with comments like this:

<!-- Hero Section -->
<h1 class="hero-title">Transform Your Business with Expert Strategy</h1>
<p class="hero-subtitle">We help forward-thinking companies achieve sustainable growth...</p>

3 Edit the Text

Replace the text between the HTML tags with your own content. For example:

<!-- Before -->
<h1 class="hero-title">Transform Your Business with Expert Strategy</h1>

<!-- After -->
<h1 class="hero-title">Your Company Name - Professional Business Solutions</h1>

4 Save and Test

Save the file and open it in your browser to see the changes. Refresh the page if it's already open.

Pro Tip: Use descriptive, benefit-focused text that speaks to your target audience. Highlight what makes your business unique.

Changing Colors

The template uses CSS variables for easy color customization. Here's the default color palette:

Primary
#2c3e50
Secondary
#3498db
Accent
#1abc9c
Light
#f8f9fa
Gray
#7f8c8d

To change colors throughout the entire template:

1 Open the CSS File

Open assets/css/style.css in your code editor.

2 Find the Color Variables

Look for the :root section at the top of the file (around line 13):

:root {
    /* Primary Colors */
    --primary-color: #2c3e50;
    --secondary-color: #3498db;
    --accent-color: #1abc9c;
    
    /* Neutral Colors */
    --light-color: #f8f9fa;
    --dark-color: #2c3e50;
    --gray-color: #7f8c8d;
    --light-gray: #ecf0f1;
}

3 Change the Color Values

Replace the hex color codes with your brand colors. For example:

/* Your Brand Colors */
--primary-color: #003366;
--secondary-color: #0066cc;
--accent-color: #00a86b;
--light-color: #f5f7fa;
--dark-color: #1a2b3c;
--gray-color: #6c757d;
--light-gray: #e9ecef;

4 Save and Refresh

Save the file and refresh your browser to see all colors updated automatically.

Important: Changing these variables will update colors across the entire template. Test your color choices to ensure good contrast for readability.

Changing Images

To replace placeholder images with your own:

1 Prepare Your Images

Optimize your images for web use. Recommended sizes:

  • Hero images: 1200px × 800px (or similar ratio)
  • Team photos: 400px × 400px (square format)
  • Service images: 800px × 600px
  • Blog images: 800px × 400px

Use tools like Photoshop, Canva, or online compressors to optimize file size.

2 Replace Image Files

Copy your images to the appropriate folders in assets/images/:

  • Hero/banner images → assets/images/hero/
  • Team member photos → assets/images/team/
  • Service-related images → assets/images/services/
  • Blog/article images → assets/images/blog/
  • Logos and icons → assets/images/logos/

3 Update Image References

In the HTML files, find image tags and update the src attribute. For example:

<!-- Before -->
<img src="assets/images/hero/hero-main.jpg" alt="Business team collaborating">

<!-- After -->
<img src="assets/images/hero/your-hero-image.jpg" alt="Your company team meeting">

4 Update Alt Text

Always update the alt attribute with descriptive text for accessibility and SEO:

<img src="assets/images/team/leader-1.jpg" alt="Sarah Johnson, CEO of Your Company">
Best Practice: Use professional, high-quality images that represent your brand. Smiling team photos build trust, while clean office images convey professionalism.

Changing Icons

The template uses Font Awesome icons. To change an icon:

1 Find the Icon You Want

Visit fontawesome.com/icons and browse or search for icons.

2 Get the Icon Class

Click on an icon and copy the class name. For example, for a chart line icon:

<i class="fas fa-chart-line"></i>

3 Replace in Your HTML

Find the icon you want to change in your HTML file and replace the class. For example:

<!-- Before -->
<div class="service-icon">
    <i class="fas fa-chart-line"></i>
</div>

<!-- After -->
<div class="service-icon">
    <i class="fas fa-chart-bar"></i>
</div>
Tip: Stick to a consistent icon style (solid "fas", regular "far", or brands "fab") throughout your site for a cohesive look.

Adding New Pages

To create additional pages based on the template:

1 Duplicate an Existing Page

Copy an existing HTML file (like about.html) and rename it (e.g., portfolio.html).

2 Update Page Content

Edit the content within the main content sections while keeping the header, navigation, and footer.

3 Update Navigation

Add a link to your new page in the navigation menu. Edit the nav section in each HTML file:

<li class="nav-item">
    <a class="nav-link" href="portfolio.html">Portfolio</a>
</li>

4 Update Page Title and Metadata

Change the <title> tag and meta description for SEO:

<title>Our Portfolio | CorporatePro Business Solutions</title>
<meta name="description" content="View our portfolio of successful business transformation projects and client case studies">

Advanced Customization

For more advanced changes, here are some pointers:

Modifying Layout

The template uses Bootstrap's grid system. To change layout structure:

  • Study Bootstrap grid documentation at getbootstrap.com
  • Adjust column classes (col-lg-6, col-md-4, etc.)
  • Maintain responsiveness by using appropriate breakpoints

Custom JavaScript

The main JavaScript file is well-commented. Key functions include:

  • Navigation and mobile menu
  • Form validation
  • Scroll animations
  • Testimonial slider

You can extend or modify these functions in assets/js/script.js.

Responsive Adjustments

For device-specific adjustments, edit responsive.css. The template uses a mobile-first approach.

Note: For complex customizations, consider hiring a professional developer if you're not comfortable with coding.

Troubleshooting Common Issues

Images Not Displaying

  • Check file paths are correct
  • Ensure image files are in the right folder
  • Verify file names match exactly (case-sensitive on some servers)

Layout Looks Broken

  • Clear browser cache (Ctrl+F5 or Cmd+Shift+R)
  • Check for missing closing HTML tags
  • Verify Bootstrap CSS is loading

Mobile View Issues

  • Check responsive.css for specific styles
  • Ensure viewport meta tag is present
  • Test on actual mobile devices when possible

JavaScript Not Working

  • Check browser console for errors (F12)
  • Verify script.js is loading
  • Ensure no syntax errors in custom code

Going Live

When you're ready to publish your website:

1 Final Testing

Test your site on different browsers and devices. Check all links and forms work correctly.

2 Optimization

Compress images, minify CSS/JS files, and enable compression on your hosting server.

3 Upload to Hosting

Upload all files to your web hosting account using FTP or your host's file manager.

4 Configure Domain

Point your domain to your hosting and set up any necessary redirects.

Congratulations! Your professional business website is now live. Remember to update your content regularly and monitor your site's performance.
