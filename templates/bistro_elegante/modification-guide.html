Bistro Elegante Template

Complete Customization Guide

Introduction

Welcome to the Bistro Elegante Restaurant Template customization guide. This comprehensive guide will walk you through every aspect of modifying the template to suit your restaurant's needs.

Before You Begin:
  • Backup the original files before making changes
  • Use a proper code editor (VS Code, Sublime Text, Atom)
  • Test changes in a local development environment first
  • Keep a changelog of modifications made

Quick Start Modifications

Step 1: Update Brand Identity

Open js/config.js and modify these sections:

// Brand Identity
brand: {
    name: 'Bistro Elegante',           // Change to your restaurant name
    tagline: 'Modern Dining Experience',
    logo: {
        light: 'assets/logo.svg',      // Update logo paths
        dark: 'assets/logo-light.svg',
    },
    colors: {
        primary: '#c17c54',            // Your main brand color
        secondary: '#8b4513',
        accent: '#d89c7c'
    }
},

Step 2: Update Contact Information

In the same js/config.js file:

// Contact Information
contact: {
    phone: '+15551234567',            // Your phone number
    formattedPhone: '(555) 123-4567',
    email: 'hello@bistroelegante.com', // Your email
    address: {
        street: '123 Gourmet Avenue',  // Your address
        city: 'Food City',
        state: 'FC',
        zip: '12345',
        full: '123 Gourmet Avenue, Food City, FC 12345'
    }
},

Step 3: Update Business Hours

// Business Hours
hours: {
    monday: { open: '17:00', close: '22:00' },
    tuesday: { open: '17:00', close: '22:00' },
    // ... update for all days
    sunday: { open: '10:00', close: '15:00' }
},

Image Replacement Guide

Required Image Sizes

  • Hero Images: 1920×800px (landscape)
  • Dish Photos: 800×600px (square or portrait)
  • Team Photos: 400×400px (square)
  • Category Images: 600×400px (landscape)

Image Optimization

  1. Use high-quality, well-lit food photography
  2. Optimize images with tools like:
    • TinyPNG for compression
    • Squoosh for WebP conversion
    • GIMP for resizing
  3. Convert to WebP format for better performance
  4. Use descriptive alt text for accessibility

Logo Creation

Creating Your Logo

  1. Design your logo in vector format (SVG recommended)
  2. Create two versions:
    • logo.svg: For light backgrounds (primary logo)
    • logo-light.svg: For dark backgrounds (footer, dark sections)
  3. Recommended logo dimensions: 200×50px
  4. Tools to use:
    • Canva (free, web-based)
    • Inkscape (free, desktop)
    • Adobe Illustrator (professional)

Menu System Customization

Adding New Menu Categories

In project-data.json, add new categories to the menu object:

{
  "menu": {
    "categories": [
      {
        "id": "new-category",
        "name": "New Category Name",
        "description": "Category description",
        "items": [
          // Add items here
        ]
      }
    ]
  }
}

Menu Item Tags

Available dietary tags for menu items:

  • "vegetarian" - Shows leaf icon
  • "vegan" - Shows seedling icon
  • "gluten-free" - Shows bread icon
  • "spicy" - Shows pepper icon
  • "healthy" - Shows apple icon

Example usage:

"tags": ["vegetarian", "gluten-free"]

Troubleshooting

Common Issues

Images Not Displaying

  • Check file paths are correct
  • Verify image files exist in assets folder
  • Ensure filenames match exactly (case-sensitive)
  • Check browser console for 404 errors

JavaScript Errors

  • Open browser Developer Tools (F12)
  • Check Console tab for error messages
  • Verify all script files are loading
  • Check for missing commas in JSON files

Layout Issues

  • Clear browser cache (Ctrl+F5 or Cmd+Shift+R)
  • Check CSS file is loading properly
  • Verify Bootstrap CDN is accessible
  • Test in incognito/private mode

Deployment Instructions

Preparing for Launch

  1. Compress all images using TinyPNG or similar
  2. Convert images to WebP format for better performance
  3. Minify CSS and JavaScript files (optional)
  4. Update all TODO markers in the code
  5. Add your Google Analytics tracking ID in js/config.js
  6. Create favicon.ico file from your logo
  7. Set up email forwarding for contact forms

Uploading to Web Host

  1. Use FTP client (FileZilla, Cyberduck) or hosting control panel
  2. Upload all files maintaining the same folder structure
  3. Set index.html as the default document
  4. Configure SSL certificate for HTTPS
  5. Set up proper file permissions (644 for files, 755 for folders)
  6. Test the live site thoroughly
