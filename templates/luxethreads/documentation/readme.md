QUICK START GUIDE
-----------------

1. Unzip the template package to your desired location
2. Open index.html in your browser to view the template
3. For development, use a code editor like VS Code
4. Customize the template by editing the HTML, CSS, and JS files
5. Replace placeholder content with your actual content
6. Update image paths to point to your images
7. Test the template across different browsers and devices
8. Deploy to your web server

CUSTOMIZATION GUIDE
-------------------

CHANGING COLORS:
The template uses CSS variables for easy color customization. Open assets/css/style.css and modify the values in the :root selector:

:root {
    --primary-dark: #1a1a1a;
    --primary-light: #f8f9fa;
    --accent-color: #d4af73;
    --accent-hover: #c19a5b;
    /* ... other variables */
}

REPLACING IMAGES:
All images are located in the assets/images/ folder. Replace them with your own images while keeping the same file names, or update the image paths in the HTML files.

EDITING PRODUCTS:
Product data is managed in assets/js/main.js. Update the sample products array with your actual products. For production, you would connect to a backend database.

MODIFYING TEXT:
Edit text content directly in the HTML files. The template uses semantic HTML structure with clear comments to help you identify sections.

ADDING NEW PAGES:
Duplicate an existing HTML file and modify the content. Update the navigation in all pages to include your new page link.

CHANGING FONTS:
The template uses Google Fonts. To change fonts, update the link in the head section and modify the font-family variables in style.css.

ANIMATION CUSTOMIZATION:
All animations are defined in assets/css/animations.css. You can adjust timing, easing, and effects in this file.

CONNECTING TO BACKEND:
The template is frontend-only. To connect to a backend:
1. Set up a server (Node.js, PHP, Python, etc.)
2. Create API endpoints for products, cart, orders
3. Update JavaScript files to fetch data from your API
4. Implement authentication and payment processing
