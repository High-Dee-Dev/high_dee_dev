Image Replacement Guidelines
  • Product Images: Use high-quality photos with clean backgrounds. Show multiple angles and details.
  • Size & Format: Optimize images for web (JPEG for photos, PNG for graphics, SVG for icons).
  • Naming: Use descriptive file names (e.g., "black-dress-front-view.jpg").
  • Alt Text: Always add descriptive alt text for accessibility and SEO.
  • Sources: Use your own photos or purchase from stock photo websites.
