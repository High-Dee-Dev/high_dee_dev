Vela Template Editing Guide

Complete instructions for customizing your fashion e-commerce template

Getting Started

This template is built with HTML5, CSS3, JavaScript, and Bootstrap 5. No build tools or preprocessors are required. All files are ready to edit directly.

1. File Structure Overview

Here's what you'll find in the template package:

fashion-bootstrap-template/
├── index.html              # Homepage
├── shop.html               # Product listing
├── product-details.html    # Single product page
├── cart.html               # Shopping cart
├── checkout.html           # Checkout process
├── about.html              # Brand story
├── contact.html            # Contact page
├── account.html            # User account
├── style.css              # Main stylesheet
├── script.js              # All JavaScript functionality
├── images/                # All image assets
└── documentation/         # This guide and README

Editing the Header & Navigation

1. Logo Replacement

Replace the logo in the navigation bar:

<!-- In every HTML file, find this line in the navigation section -->
<a class="navbar-brand" href="index.html">
    <img src="images/brand-logo.svg" alt="Vela" height="32">
</a>

<!-- Replace with your logo -->
<a class="navbar-brand" href="index.html">
    <img src="images/your-logo.png" alt="Your Brand" height="40">
</a>

Recommended logo dimensions: minimum 160px width, SVG or PNG format.

2. Navigation Links

Update the navigation menu items in all HTML files:

<ul class="navbar-nav mx-auto">
    <li class="nav-item">
        <a class="nav-link" href="index.html">Home</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="shop.html">Shop</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="about.html">Our Story</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="contact.html">Contact</a>
    </li>
</ul>

Add or remove menu items as needed. The "active" class indicates the current page.

Updating Products

1. Adding New Products

Products are managed in the script.js file. Find the products array:

const products = [
    {
        id: 1,
        name: 'Linen Relaxed Blazer',
        category: 'outerwear',
        price: 248,
        image: 'images/product-01.jpg',
        color: 'Natural',
        material: 'Linen',
        description: 'Unlined blazer in breathable European linen...',
        featured: true
    },
    // Add more products here
];

Add a new product object following the same structure. Make sure each product has a unique ID.

2. Product Images

Add your product images to the images/ folder. Update the image path in the product object:

{
    id: 7,
    name: 'Your Product Name',
    category: 'category',
    price: 199,
    image: 'images/your-product-image.jpg',  // Update this
    // ... other properties
}
Image Guidelines:
• Recommended size: 800x1000px for product images
• Use consistent aspect ratios
• Optimize images for web (compress before uploading)
• Include multiple angles for product detail pages

3. Product Categories & Filters

Update the filter sidebar in shop.html to match your product categories:

Customizing Colors

1. Primary Color Scheme

Edit the CSS variables at the top of style.css:

:root {
    --font-primary: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;
    --font-secondary: 'Playfair Display', serif;
    --color-dark: #1a1a1a;      /* Main text and dark elements */
    --color-light: #f8f9fa;     /* Light backgrounds */
    --color-gray: #6c757d;      /* Secondary text */
    --color-border: #e9ecef;    /* Borders and dividers */
    --transition-base: all 0.2s ease;
}

Change the hex values to your brand colors. The template uses these variables consistently throughout.

2. Button Colors

Bootstrap button classes are used throughout. To change button colors globally:

/* Add to the end of style.css */
.btn-dark {
    background-color: #your-color;
    border-color: #your-color;
}

.btn-dark:hover {
    background-color: #your-darkened-color;
    border-color: #your-darkened-color;
}

.btn-outline-dark {
    color: #your-color;
    border-color: #your-color;
}

.btn-outline-dark:hover {
    background-color: #your-color;
    border-color: #your-color;
}

Changing Fonts

1. Google Fonts Replacement

In the <head> of each HTML file, update the Google Fonts link:

<!-- Current fonts -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&family=Playfair+Display:wght@400;500;600&display=swap" rel="stylesheet">

<!-- Replace with your chosen fonts -->
<link href="https://fonts.googleapis.com/css2?family=Your+Font:wght@400;500;600&family=Your+Secondary+Font&display=swap" rel="stylesheet">

Then update the CSS variables in style.css:

:root {
    --font-primary: 'Your Font', sans-serif;
    --font-secondary: 'Your Secondary Font', serif;
    /* ... other variables */
}

Cart & Checkout Customization

1. Shipping Settings

Update shipping information in cart.html and checkout.html:

<!-- Find and update this text -->
<p class="small text-muted">Free shipping on orders over $200. Taxes and shipping calculated at checkout.</p>

<!-- Update the amount -->
<p class="small text-muted">Free shipping on orders over $[your-amount]. Taxes and shipping calculated at checkout.</p>

2. Payment Methods

Replace the payment methods image in the footer:

<!-- In every HTML file footer -->
<img src="images/payment-methods.png" alt="Accepted payment methods" height="24">

<!-- Replace with your payment methods graphic -->
<img src="images/your-payment-methods.png" alt="We accept Visa, Mastercard, etc." height="30">

Create a graphic showing your accepted payment methods (recommended width: 200px).

3. Tax Calculation

The template shows static tax amounts. For real tax calculation, you'll need to:

  1. Integrate with a tax calculation API (like TaxJar or Avalara)
  2. Update the updateCartTotals() function in script.js
  3. Implement server-side validation for checkout
Important: This template provides front-end functionality only. For production use, you must implement proper backend validation, payment processing, and tax calculation.

Advanced Customizations

1. Adding New Pages

To add a new page:

  1. Duplicate an existing HTML file (like about.html)
  2. Rename it (e.g., lookbook.html)
  3. Update the navigation links in all files to include the new page
  4. Customize the content as needed

2. Integrating with E-commerce Platforms

This template is front-end only. To connect to a backend:

Option A: Shopify
  • Convert HTML/CSS to Shopify Liquid theme
  • Replace product data with Shopify product loops
  • Use Shopify's cart and checkout system
Option B: WooCommerce
  • Convert to WordPress theme
  • Implement WooCommerce template files
  • Use WooCommerce hooks for cart functionality
Option C: Custom Backend
  • Set up a database for products and orders
  • Convert JavaScript functions to API calls
  • Implement secure payment processing

3. Performance Optimization

Before launching your site:

  1. Compress all images using tools like TinyPNG or Squoosh
  2. Minify CSS and JavaScript files
  3. Implement lazy loading for images below the fold
  4. Set up proper caching headers
  5. Use a CDN for static assets

Troubleshooting

Common Issues

Images not loading
  • Check file paths are correct
  • Ensure image filenames match exactly (case-sensitive on some servers)
  • Verify images are in the correct folder
JavaScript not working
  • Check browser console for errors (F12 → Console)
  • Ensure script.js is loaded after jQuery and Bootstrap
  • Check for missing semicolons or syntax errors
Responsive issues
  • Test on actual mobile devices, not just browser emulation
  • Check that viewport meta tag is present
  • Verify Bootstrap CSS is properly loaded

Getting Help

If you encounter issues not covered in this guide:

  1. Check the browser console for error messages
  2. Validate your HTML at validator.w3.org
  3. Search for Bootstrap 5 documentation
  4. Consult with a front-end developer for complex customizations
